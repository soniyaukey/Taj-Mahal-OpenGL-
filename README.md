# 🕌 Taj Mahal OpenGL Project

A 3D graphical representation of the **Taj Mahal** developed using OpenGL and C++. This project demonstrates fundamental computer graphics concepts such as rendering, object modeling, transformations, and visualization techniques. The model recreates the architectural structure of the Taj Mahal and provides an interactive environment for understanding graphics programming.

---

## ✨ Features

* 3D model of the Taj Mahal
* Realistic object rendering using OpenGL
* Implementation of geometric transformations
* Demonstration of computer graphics concepts
* Interactive visualization environment
* Educational project for graphics programming

---

## 🛠️ Technologies Used

* **C++**
* **OpenGL**
* **Computer Graphics**
* **GLUT (OpenGL Utility Toolkit)**

## 🚀 How to Run

1. Install a C++ compiler (GCC/MinGW or Visual Studio).
2. Install OpenGL and GLUT libraries.
3. Compile the source code:

```bash
g++ main.cpp -o tajmahal -lGL -lGLU -lglut
```

4. Run the executable:

```bash
./tajmahal
```
